## 🚀 Key Features:
 - 🎯 **Fine-Grained Reactivity**: Powered by `Signal`s. Changing a value updates a specific `TextNode`, or `Attribute`, not the entire component.
 - 🌍 **Platform Agnostic**: The Core (The Brain) is decoupled from the Renderer. Render to the *DOM*, *SSR* *strings*, *Canvas*, or the *Terminal*.
- ⚡ **O(1) Lookups**: Integrated registry system for instant element access without expensive DOM querying.
